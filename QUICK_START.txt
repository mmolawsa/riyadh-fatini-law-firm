🚀 البدء السريع:

1️⃣ التشغيل المحلي:
   pnpm install
   pnpm dev
   ثم افتح: http://localhost:5173

2️⃣ النشر على Vercel:
   - أنشئ مستودع GitHub
   - اربطه بـ Vercel
   - سيتم النشر تلقائياً

3️⃣ ربط النطاق:
   - أضف lawyerriyadhfatini.com في Vercel
   - حدّث DNS عند مسجل النطاق

📚 للمزيد من التفاصيل:
   اقرأ ملف SETUP_INSTRUCTIONS.md
